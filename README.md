# Telegram-бот для стоматологии
Готовый Telegram-бот для стоматологических клиник. Пациент смотрит услуги, выбирает врача, записывается на приём.
**Демо:** [@dental_smile_demo_bot](https://t.me/dental_smile_demo_bot)
## Возможности
- Каталог услуг с фото и ценами (17 услуг)
- Список врачей
- Онлайн-запись на приём
- Акции и информация о клинике
- Отзывы с уведомлением администратору
## Стек
Python 3.10+, python-telegram-bot v20+, ConversationHandler
## Быстрый старт
1. `pip install -r requirements.txt`
2. Вставить токен в config.py
3. `python bot.py`
